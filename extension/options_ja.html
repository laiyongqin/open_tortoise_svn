一般設定

インストール

事前に、下記のインストールが必要です。

  1. TortoiseSVN をダウンロードページからダウンロードしてインストールしてください。
    既にインストール済みの方は不要です。
  2. open_tortoise_svn_host.zip をダウンロード、展開し、install.bat を実行してインストールしてください。
    このツールは、Chrome と連携して TortoiseSVN を起動します。

TortoiseProc.exe のパス

TortoiseProc.exeのパスを指定してください。

Subversion リポジトリの URL

Subversion リポジトリの URL を追加してください。

リストに追加された URL をクリックすると、TortoiseSVN が自動で立ち上がります。

例: "http://svn_repos/" を追加し、"http://svn_repos/dir/test.cpp" をクリックすると、このファイルが TortoiseSVN で開かれます。

    Open TortoiseSVN について

    概要

    この拡張機能を用いると、TortoiseSVNを直接開くことができます。
    登録された URL をクリックしたときに、TortoiseSVN リポジトリブラウザが自動で開かれます。

    この拡張機能は、右クリックメニューに TortoiseSVN のリポジトリブラウザ、ログ、注釈履歴を追加します。

    この拡張機能は、Firefox 向けに開発したOpen TortoiseSVN アドオンの Google Chrome™ 版です。

    開発者向け

    HTML ソースを編集し、rel 属性を以下のように追加すると、TortoiseSVN のログ、注釈履歴を表示できます。

    HTML5 では、rel 属性の代わりに data-tsvn-info を使うこともできます。

    • TortoiseSVN ログを開く場合 
      <a href="http://svn_repos/" rel="tsvn[log]">Open Log Viewer</a>
    • TortoiseSVN で、特定のリビジョンのログを開く場合 
      <a href="http://svn_repos/" rel="tsvn[log][12,8]">Open Log Viewer</a>
    • TortoiseSVN 注釈履歴を開く場合 
      <a href="http://svn_repos/" rel="tsvn[blame]">Open Blame Viewer</a>
