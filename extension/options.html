General Settings

Installation

Install the following tools.

  1. Download TortoiseSVN from Downloads page, then install it.
    If you have already installed it, skip this step.
  2. Download open_tortoise_svn_host.zip and extract it.
    Run install.bat to install it.
    This is a bridge tool between Chrome and TortoiseSVN.

Path of TortoiseProc.exe

Set path of TortoiseProc.exe.

URL list of Subversion repositories

Add URLs of Subversion repositories.

When you click one of the registered URL, TortoiseSVN is opened automatically.

e.g. If you add "http://svn_repos/" and click "http://svn_repos/dir/test.cpp", this file is opened in TortoiseSVN.

    About Open TortoiseSVN

    Overview

    This extension enables you to open TortoiseSVN directly.
    When you click the link to one of the registered URLs, TortoiseSVN Repository Browser is opened automatically.

    This extension also provides some context menus to open TortoiseSVN Repository Browser, Log Viewer and Blame Viewer.

    This extension is a Google Chromeâ„¢ version of Open TortoiseSVN add-on that I developed for Firefox.

    For Developers

    If you modify your HTML source and add rel attribute, you can open TortoiseSVN Log or Blame viewer as follows.

    You can also use data-tsvn-info on HTML5 instead of rel attribute.

    • To open TortoiseSVN Log viewer 
      <a href="http://svn_repos/" rel="tsvn[log]">Open Log Viewer</a>
    • To open TortoiseSVN Log of specific revision 
      <a href="http://svn_repos/" rel="tsvn[log][12,8]">Open Log Viewer</a>
    • To open TortoiseSVN Blame viewer 
      <a href="http://svn_repos/" rel="tsvn[blame]">Open Blame Viewer</a>
