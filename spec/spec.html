Example

Prepare

  1. Set "Path to TortoiseProc.exe".
  2. Add a URL "https://github.com/masamitsu-murase/open_tortoise_svn.git/trunk".
  3. Set default action to "Repository Browser".
  4. Add extension actions.
    • "*.md" as "Log"
    • "*.json, *.js" as "Blame"
    • "*.cpp" as "Open in Chrome"
  5. Check "A help tip".

Check URL and extension action

  • /trunk
  • /trunk/README.md
  • /trunk/extension/manifest.json
  • /trunk/native_messaging/open_tortoise_svn_host.cpp
  • https://www.google.co.jp/

Check attribute

  • /trunk, tsvn[log]
  • /trunk, tsvn[log][2]
  • /trunk, tsvn[log][10,8]
  • /trunk/README.md, tsvn[blame]
  • /trunk/extension/manifest.json, tsvn[open_in_browser]
  • /trunk/native_messaging/open_tortoise_svn_host.cpp, tsvn[browser]
